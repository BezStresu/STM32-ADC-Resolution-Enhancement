Improved the effective number of bits (ENOB) of an STM32 microcontroller ADC for low-amplitude AC voltage measurements by implementing controlled dithering using a triangular waveform added to the input signal prior to sampling and quantization.

The method linearized the quantization process and reduced deterministic quantization errors, enabling higher measurement resolution through oversampling and filtering. The project applied techniques from STMicroelectronics application note AN5537 and demonstrated practical skills in ADC signal conditioning, embedded system measurement optimization, and advanced analog-to-digital conversion techniques.
